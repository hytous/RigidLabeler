# RigidLabeler Data - Images Directory

This directory contains input images for registration labeling.

## Structure

Place your image pairs here:
- Fixed (base) images
- Moving (warp) images

Example:
```
images/
├── vis_001.png    # Visible light image
├── ir_001.png     # Infrared image
├── vis_002.png
├── ir_002.png
└── ...
```
